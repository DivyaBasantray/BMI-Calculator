# 📊 BMI Calculator (Python)

A simple and user-friendly **BMI (Body Mass Index) Calculator** built using Python. This project takes user input for height and weight, calculates BMI, and categorizes the result based on standard health ranges.

---

## 🚀 Features

- 🧮 Calculates BMI using user input  
- 📏 Supports height and weight input  
- 📊 Displays BMI value with category  
- ✅ Simple logic, beginner-friendly  
- 💻 Command-line based interface  

---

## 🧠 How It Works

BMI is calculated using the formula:

\[
BMI = \frac{weight (kg)}{height (m)^2}
\]

Based on the calculated BMI, the program classifies the result into:

- Underweight  
- Normal weight  
- Overweight  
- Obese
